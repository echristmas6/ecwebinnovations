Studdy

Studdy is a personalized study platform built with Django and deployed on PythonAnywhere. It leverages a ChatGPT API to generate custom practice problems for various high school subjects. The site offers: AI-Generated Practice Problems: Practice problems generated by a ChatGPT integration to test your knowledge. Subject-Specific Resources: Helpful resources to deepen your understanding. User Authentication: User login/sign-up to track progress. Technologies: Django, Python, ChatGPT API, HTML, CSS, PythonAnywhere.
